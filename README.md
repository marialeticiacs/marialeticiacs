# 🌸 Olá, eu sou a Maria Letícia!

Com base em Big Data, aplico uma visão sistêmica para automatizar jornadas de deployment e garantir a integridade dos dados na nuvem. Transformo ambientes complexos em arquiteturas de código limpas, seguras e prontas para escalar.

### 🚀 O que move meu trabalho
* **Arquitetura Cloud & DevOps:** Foco em design de infraestruturas resilientes e pipelines de CI/CD automatizados (Azure/OCI).
* **DevSecOps:** Segurança como pilar central, do código em repouso até a esteira de deployment.
